# Deploy this repo on Vercel

1. Go to **https://vercel.com/new** and import **sundaramCODERdubey/Sundaram_website**.

2. **Important:** Set **Root Directory** to `college-society-app` (click Edit next to it).

3. Leave Framework Preset as **Vite**. Click **Deploy**.

The app will be built from `college-society-app` and client-side routes will work with the included rewrites.
